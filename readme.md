# GRAVITAS — Cosmic Sandbox

**"Play God. Wreck Worlds. Watch the Beauty."**

Gravitas is a minimalist yet stunning gravity physics sandbox designed for wonder and relaxation. Built as a high-performance standalone web experience, it allows you to create solar systems, observe gravitational dances, and explore the reach of cosmic forces with intuitive controls.

## 🚀 Features

- **Single-File Portability**: The entire game (HTML5, CSS3, JS) is contained in a single, highly-optimized `.html` file.
- **Premium Aesthetics**: Featuring modern glassmorphism, Google Fonts typography (Outfit & Syncopate), and high-DPI (Retina) support.
- **Robust Physics Engine**:
  - N-body gravitational simulation with custom Vector math.
  - Sub-stepping integrator for high-fidelity stable orbits.
  - Softened gravity to prevent collision explosions.
  - Collision shattering with glowing particle effects.
- **Advanced Camera System**:
  - Smooth pan and zoom (scroll wheel or two-finger touch).
  - **Focus Tracking**: Lock the camera onto the most massive body to follow its journey.
  - **Cinematic Mode**: Hide the UI for an immersive, distraction-free view.
- **Interactive God Mode**:
  - Drag-to-launch mechanics for precise velocity control.
  - Multiple body types: Stars, Planets, Asteroids, and Voids (Black Holes).
  - Real-time adjustment of Gravity strength and Time scale.
- **Mobile First**: Fully responsive HUD and multi-touch support for iPad and mobile play.
- **Cosmos State**: Save and Load your favorite systems locally.

## 🛠️ Technology Stack

- **Graphics**: HTML5 Canvas API (optimized for 60FPS).
- **Physics**: Custom JavaScript N-Body Engine.
- **Styling**: Vanilla CSS3 with Flexbox/Grid and Backdrop-Filters.
- **Audio**: Web Audio API for procedural cosmic ambient hum.
- **Typography**: Google Fonts (Outfit, Syncopate).

## 🎮 How to Play

1.  **Open** `gravitas.html` in any modern web browser.
2.  **Launch**: Click and drag anywhere to aim and launch a celestial body.
3.  **Explore**:
    - **Middle Click / Two-Finger Drag**: Pan the camera.
    - **Scroll / Pinch**: Zoom in/out.
    - **Presets**: Use the top-left menu to quickly load stable Solar Systems or Chaos modes.
4.  **Polish**: Use the **Eye** icon to enter Cinematic Mode or the **Focus** icon to track your primary star.

## 🌌 Keyboard Shortcuts

- `Space`: Pause/Resume
- `1-4`: Select Body Type (Star, Planet, Asteroid, Void)
- `C`: Clear Universe
- `T`: Toggle Trails
- `F`: Toggle Focus (on most massive body)
- `V`: Toggle Cinematic Mode
